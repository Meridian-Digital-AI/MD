# Meridian Digital

AI-powered digital agency website built with Next.js 14, TypeScript, and Tailwind CSS.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Fonts:** Sora (headings) + Inter (body)

## Project Structure

```
src/
  app/           — Pages and routes
  components/    — Reusable UI components
  lib/data/      — Content and data modules
```

## Pages

- `/` — Homepage
- `/services` — What We Build (pricing tiers)
- `/work` — Portfolio / demos
- `/how-it-works` — Process + FAQ
- `/about` — About us
- `/contact` — Contact form + booking
- `/sectors/*` — 5 sector landing pages

## Deployment

Deploy to Vercel:

```bash
npm run build
```
